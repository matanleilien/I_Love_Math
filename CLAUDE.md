# I Love Math - Game Project

## Deployment
- Hosted on Vercel (project: `i-love-math`)
- Deploy command: `vercel --yes --prod`
- Live URL: https://i-love-math-hazel.vercel.app

## Workflow Rules
- **Always git commit alongside every Vercel deploy.** Every iteration of changes should have a matching commit with a descriptive message before or alongside the deploy.
- **After every commit, ask the user if they want to push to master.** Do not push automatically.
